# git-lex-kit-squad

Multi-agent collaboration kit for [git-lex](https://github.com/repolex-ai/git-lex).

Team coordination across carbon and silicon agents.

## Document Types

- **Agent** — squad member identity (human or AI)
- **Message** — communications between agents
- **Decision** — group decisions with rationale
- **Task** — shared work items with assignments
- **Project** — project tracking
- **Discovery** — shared findings
- **Note** — freeform team knowledge

## Install

```bash
git lex init --kit squad
```
